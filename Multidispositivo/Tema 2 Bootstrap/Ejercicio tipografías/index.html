Con la etiqueta "mark" podemos resaltar texto. También podemos marcar texto como eliminado o insertado, útil para mostrar anotaciones o versiones de un documento. El subrayado tradicional se delimita con la etiqueta "u". Para negrillas, utilizamos "strong", y para itálica, "em".

Podemos utilizar clases CSS incluidas en Bootstrap para alinear texto:

Centrado, "text-center"

A la derecha, "text-right"

Bootstrap incluye clases CSS para transformar la capitalización (mayúsculas/minúsculas):

Mostrar texto en minúsculas: "text-lowercase". Mostrar texto en mayúsculas: "text-uppercase". Mostrar texto con inicial en mayúsculas: "text-capitalize".

Podemos usar "address" para indicar información de contacto sobre un objeto o el documento completo:

Para delimitar texto citado, podemos utilizar "blockquote" alrededor de cualquier contenido HTML. Usualmente es un párrafo delimitado con la etiqueta "p":/

En algún lugar de la mancha, de cuyo nombre no quiero acordarme...

También podemos hacer referencia a la fuente o autor de una cita con un "footer" dentro de la misma:
